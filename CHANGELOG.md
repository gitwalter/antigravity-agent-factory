# Changelog

All notable changes to the Antigravity Agent Factory project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-08

### Added
- Initial release of Antigravity Agent Factory
- 12 factory agents for orchestrating development workflows
- 37 factory skills for specialized procedures
- 34 technology stack blueprints (Python, TypeScript, Java, SAP, Blockchain, etc.)
- 100+ reusable patterns for axioms, principles, methodologies, and workflows
- 100+ knowledge files with domain expertise in JSON format
- 200+ code and document templates
- Integrity Guardian (Layer 0 protection system)
- Memory System with user-validated learning
- Formal verification system using Lean 4
- Agent Society Verification for multi-agent coordination
- Knowledge Evolution System for keeping blueprints current
- Team Workshop System with collaborative games
- Complete test suite with unit, integration, and validation tests
- CI/CD pipeline with automated validation

### Changed
- Updated all branding from "Cursor Agent Factory" to "Antigravity Agent Factory"
- Fixed project structure documentation in README.md
- Corrected git executable path configuration in `safe_commit.py`
- Updated validation scripts to use correct configuration keys

### Fixed
- JSON syntax errors in knowledge files (`rag-patterns.json`, `langchain-patterns.json`)
- Syntax errors in validation scripts
- Git path resolution using `fallback_paths` from `tools.json`
- Test assertions to match Antigravity branding
- Tree formatting in project structure visualization
