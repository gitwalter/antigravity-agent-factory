## How to Use

1. **Choose a workshop** based on your learning goals
2. **Check prerequisites** in the workshop document
3. **Follow the phases** (Concept → Demo → Exercise → Challenge → Reflection)
4. **Complete the self-assessment** to verify your understanding
5. **Move to the next workshop** in the learning path

## Workshop Structure

Each 2.5-hour workshop follows the same proven structure:

| Phase | Duration | Purpose |
|-------|----------|---------|
| **Concept** | 30 min | Theory, architecture, mental models |
| **Demo** | 30 min | Live coding walkthrough |
| **Exercise** | 45 min | Guided hands-on practice |
| **Challenge** | 30 min | Independent problem-solving |
| **Reflection** | 15 min | Key takeaways, self-assessment |
