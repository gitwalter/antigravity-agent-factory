# Trust Tier Selection Guide

Antigravity uses a tiered trust system to verify agent reliability and security.

## Trust Tiers

### Tier 1: Unverified
- **Description:** Basic agents with no verification.
- **Use Cases:** Simple tasks, prototypes.

### Tier 2: Community Verified
- **Description:** Agents reviewed by the community.
- **Use Cases:** Non-critical applications.

### Tier 3: Officially Verified
- **Description:** Agents passing rigorous security and functionality audits.
- **Use Cases:** Critical infrastructure, financial transactions.

## Selection Process

Use the Trust Tier Decision Matrix to determine the appropriate tier for your agent.
