# Society Integration Guide

This guide details how to integrate your agents into the Antigravity Agent Society.

## Integration Steps

1. **Protocol Compliance:** Ensure your agent adheres to the Agent Society Protocol (ASP).
2. **Registration:** Register your agent with the Society Registry.
3. **Trust Tier Selection:** Choose an appropriate trust tier based on your agent's capabilities and verification needs.

## Required Skills

- `agent-coordination`
- `verified-communication`
