# Agent Development

Multi-step workflow for developing AI agents from design through deployment. Covers agent definition, tool binding, state management, testing, and production deployment.

**Version:** 1.0.0  
**Created:** 2026-02-10  
**Applies To:** langgraph-agent, langchain-agent

## Trigger Conditions

This workflow is activated when:

- New agent system needed
- Agent design requested
- User asks to "build an agent" or "create an AI agent"
- Tool-using LLM application required

**Trigger Examples:**
- "Build an agent that can search and summarize"
- "Create a customer support agent with tools"
- "Develop an agent for document processing"
- "Implement a LangGraph agent with state"
