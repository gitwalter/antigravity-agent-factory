# SAP Documentation MCP Setup Guide

Fast search for unified SAP documentation (SAPUI5, CAP, OpenUI5, wdi5) with BM25 full-text search.

## Installation
```powershell
npx -y mcp-sap-docs
```

## Tools
- `search`: Search across official SAP documentation
- `fetch`: Retrieve document content or snippets
- `sap_community_search`: Real-time SAP Community content
- `sap_help_search`: SAP Help Portal documentation search
- `sap_help_get`: Retrieve complete SAP Help pages

## Configuration
No authentication is required for basic public search operations.
