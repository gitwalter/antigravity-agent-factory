# DeepWiki MCP Setup Guide

AI-powered web research and content processing via DeepWiki.

## Configuration
This is an SSE-based MCP. Add to your `mcp_config.json`:

```json
"deepwiki": {
  "serverUrl": "https://mcp.deepwiki.com/mcp"
}
```

## Tools
- `search`: AI-powered web research and content retrieval.
